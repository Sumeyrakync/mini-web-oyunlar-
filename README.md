# 🎮 Mini Web Games Collection

Welcome to my mini web games repository! This project features a collection of fun, single-file, browser-based games built entirely with vanilla web technologies.

**🌟 Play the games here:** [Live Demo](https://Sumeyrakync.github.io/mini-web-oyunlari/) *(Click to play!)*

---

## 🕹️ The Games

### 🍔 1. Hamburger Train (Snake Variant)
A delicious twist on the classic Snake game. Collect ingredients (🍅, 🥩, 🥬, 🧀, 🍟, 🥤) to build the ultimate burger. 
* **Features:** Retro 8-bit sound effects generated via the Web Audio API, pixelated canvas styling, and safe-food generation algorithms.

### 🕺⚡ 2. Speeding Halay Simulator
A unique, musical take on the classic formula. Instead of a snake, you control a line-dance (Halay) leader! Gather people to grow your dancing crew. 
* **The Catch:** Every time a new person joins the line, the game speed increases! 
* **Features:** Integrated YouTube background music (YouTube Iframe API), dynamic speed scaling, and responsive emoji rotation based on direction.

---

## ⌨️ How to Play

Both games share the same simple controls:
* **Desktop:** Use the **Arrow Keys** (Up, Down, Left, Right) to navigate.
* **Mobile/Tablet:** Use the on-screen D-pad buttons to control the direction. 

---

## 🛠️ Technologies Used

* **HTML5 Canvas:** For rendering the game grid, characters, and items.
* **Vanilla JavaScript (ES6):** For game logic, collision detection, and rendering loops (`requestAnimationFrame`).
* **CSS3:** For styling, flexbox/grid layouts, responsive design, and animations.
* **Web Audio API:** For synthesizing real-time sound effects (Hamburger Train).
* **YouTube Iframe API:** For background music integration (Halay Simulator).

---

## 🚀 How to Run Locally

You don't need any complex build tools or servers to run these games. 
1. Clone or download this repository.
2. Double-click on `index.html` to open the main menu in your default web browser.
3. Select a game and enjoy!
